## What it does
- Stores and loads todo list from local storage
- Implements basic CRUD operation
  - Create new todos
  - Mark todos as done
  - Edit existing todos
  - Delete todos
- Has a simple filter to filter by "all", "open" and "completed" todos

## Frameworks / libraries used
- React 18.2
- TailwindCSS 3.3.3
- TailwindCSS Forms 0.5.4
